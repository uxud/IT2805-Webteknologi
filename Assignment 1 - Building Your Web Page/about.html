Questions

  1. The server delivers services/resources to the client(s), such as processing power, files and data, programs, network services and printer services.
  2. The client is on the user side, and makes use of the resources the server provides.
  3. IP stands for Internet Protocol, and the IP address is a unique number that makes it possible for communication on the internet by giving your computer and address so that when you request data, the other end know where to send it to. Your IP address is given to your computer by your network provider, and it changes with your network. It is not static.
  4. DNS stands for Domain Name System which connects a domain name with the correct IP-address It works like a phone book, connecting the human-readable name of a website like www.ntnu.no to the correct machine-readable IP address.
  5. The WWW (World Wide Web) is a portion of the Internet, but they are not the same. The Internet is a networking infrastructure, a network of networks which connects millions of computers all over the world. Any computer can communicate with any other computer if they are both connected to the internet. On the Internet, protocols are used as a standard for communication. The WWW allows one to access information over the internet, and is based on the HTTP protocol to transmit data. It is built on top of the Internet to allow for information-sharing. The WWW is often accessed through browsers to access web pages that are connected by hyperlinks.
  6. HTML is short for Hyper Text Markup Language and is used when making web pages. HTML defines the structure and layout of a web page using HTML tags and elements. Browsers use HTML to render the content of the page. It uses HTML elements as building blocks.
